Key Features Of HTML 5

Feature No 1 : Semantic Elements

When constructing an HTML script, the term “semantic” refers to the meaning of the text. Semantics, to put it simply, is the use of HTML tags that accurately reflect the content they contain. For instance, using the < div > tag doesn’t suggest the content it will carry, however using the <p > tag suggests it can be used to hold paragraph information.

There are a lot of reasons why you should write Semantic tags instead of normal HTML tags, to leverage SEO, accessibility, and browser compatibility.

Feature No 2 : Video Support

HTML5 introduced the < code > < video > < /code > element for embedding videos directly in web pages.

Feature 3 : Form Validation

HTML5 offers built-in form validation, reducing the need for custom JavaScript code.
